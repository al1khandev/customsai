# Интеграция Telegram Бота

Приложение теперь поддерживает **Telegram** в дополнение к WhatsApp!

## 🤖 Как создать Telegram бота

### Шаг 1: Откройте BotFather в Telegram

1. Откройте Telegram
2. Найдите чат **@BotFather**
3. Напишите: `/newbot`
4. BotFather попросит имя бота - введите: `CustomsAIBot`
5. Укажите username: `customsai_bot` (или уникальное имя)

### Шаг 2: Получите TOKEN

BotFather вернет что-то типа:
```
Use this token to access the HTTP API:
123456789:ABCDefGhIjKlMnOpQrStUvWxYz_1A2B3C4D5E
```

Скопируйте этот TOKEN (весь, с двоеточием)

### Шаг 3: Сохранить TOKEN в переменные окружения

**Локально:**
Добавьте в `.env`:
```
TELEGRAM_BOT_TOKEN=123456789:ABCDefGhIjKlMnOpQrStUvWxYz_1A2B3C4D5E
```

**На Railway:**
1. Откройте проект Railway
2. **Variables** → добавьте:
   ```
   TELEGRAM_BOT_TOKEN=123456789:ABCDefGhIjKlMnOpQrStUvWxYz_1A2B3C4D5E
   ```
3. Нажмите **Redeploy**

### Шаг 4: Найти вашего бота в Telegram

1. Откройте Telegram
2. Найдите по username: `@customsai_bot` (или ваше имя)
3. Нажмите **Start**

## 💬 Как использовать

### Отправить данные о товарах

Напишите боту детали таможенной декларации:
```
Компания: ALDAU TRADE
БИН: 241040012345
Отправитель: QINGDAO SUNRISE
Страна: Китай
Товар: Электроника
Цена: 45000 USD
Вес: 150 кг
```

Бот автоматически анализирует и генерирует PDF!

### Или используйте кодовое слово

Если в сообщении есть кодовое слово (по умолчанию "декларация 777"), бот сразу обработает.

## ✅ Статус бота

Проверить подключение Telegram:
```
https://ВАШ_URL/status
```

Ответ должен включать:
```json
{
  "telegram_connected": true,
  "telegram_enabled": true
}
```

## 📊 Логирование

В логах Railway вы увидите:
```
✅ Telegram бот инициализирован
💬 Telegram сообщение от username: ...
✅ PDF отправлен в Telegram
```

## ⚙️ Настройки

### Минимальная длина сообщения

По умолчанию - 50 символов. Либо:
- Используйте кодовое слово
- Напишите более длинное сообщение

### Язык

Все сообщения бота на русском языке.

## 🔐 Безопасность

- TOKEN хранится в переменных окружения (защищено)
- Данные обрабатываются приватно
- WhatsApp и Telegram работают независимо

## 📱 WhatsApp + Telegram одновременно

Обе платформы работают параллельно:
- WhatsApp - для сканирования QR
- Telegram - для отправки сообщений в бот

Выбирайте удобный способ!

## 🆘 Если бот не отвечает

1. Проверьте TOKEN в переменных Railway
2. Нажмите **Redeploy**
3. Проверьте логи (вкладка Logs)
4. Попробуйте написать `/start` боту

## 📚 Дополнительные команды BotFather

```
/setcommands    - добавить команды боту
/setdescription - описание бота
/setuserpic     - установить аватар
/setdefault_administrator_rights - права администратора
```
